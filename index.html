애드립 SDK 소개

애드립 SDK 는 특정 플랫폼에 의존적이지 않으며,
원하는대로 자신의 앱에 최적화하여 다양한 광고 플랫폼을 손쉽게 적용할 수 있도록 제작되었습니다.

현재 애드립에서 지원중인 광고 플랫폼 정보

애드립에서 사용가능한 플랫폼은 아래와 같습니다.

일반 플랫폼
 * confirmed compatible with ad@m SDK 2.2.0
 * confirmed compatible with admob SDK 6.5.0
 * confirmed compatible with inmobi SDK 3.7.0
 * confirmed compatible with cauly SDK 3.0.2
 * confirmed compatible with t-ad SDK 3.1.0.6
 * confirmed compatible with naverAdPost SDK 1.2.0
 * confirmed compatible with shallWeAd SDK 2.4.7
 * confirmed compatible with millennialMedia SDK 5.1.0

사용자의 편의에 의해 각 플랫폼 사이트에서 SDK 를 별도로 다운로드 받아 연동할 수 있으며, 정상적인 동작이 확인된 SDK 버전은 위와 같습니다.

실제로 각 플랫폼의 마이너 버전 업데이트가 일어나더라도 애드립 구현부의 변경없이 업데이트 된 SDK 만 새롭게 적용하면 그대로 사용가능하며, 최소한의 구현부 수정으로 큰 수정없이 적용할 수 있도록 구성되었습니다.

애드립을 발전시켜주세요

애드립은 열린 플랫폼으로 각 광고 플랫폼 구현부를 손쉽게 커스터마이징 할 수 있도록 제작되었습니다.
애드립 구현부는 github 를 통해 자유롭게 공유가 가능합니다. 애드립을 더 쓸모있는 플랫폼으로 만들기 위해 여러분의 참여를 기다립니다.
